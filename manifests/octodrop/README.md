# Octodrop

Current Octodrop packs are split into:

- `billing/`
- `work_management/`
- `outreach/`
- `octo_ai/`

Run the full curated stack:
- `python3 manifests/octodrop/install_all.py`
- `python3 manifests/octodrop/seed_dummy_examples.py`

Or run the scripts inside each subfolder for a narrower install.
