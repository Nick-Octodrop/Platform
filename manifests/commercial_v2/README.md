# Commercial V2

Commercial quoting, orders, purchasing, invoicing, and admin configuration pack.

Contents:
- `dashboard.json`
- `contacts.json`
- `products.json`
- `quotes.json`
- `orders.json`
- `purchase_orders.json`
- `invoices.json`
- `settings.json`

Scripts:
- `python3 manifests/commercial_v2/install_all.py`
- `python3 manifests/commercial_v2/seed_dummy_examples.py`
- `python3 manifests/commercial_v2/setup_access_profiles.py`

Environment:
- `OCTO_BASE_URL`
- `OCTO_API_TOKEN`
- `OCTO_WORKSPACE_ID`
